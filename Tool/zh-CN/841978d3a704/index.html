何时使用 git fetch 和 git pull

  • 使用 git pull：

    • 想要快速同步远程仓库的更新到当前分支。
    • 不担心合并冲突或已经处理了所有潜在冲突。

  • 使用 git fetch：

    • 想要查看远程仓库的更新但不希望立即合并。
    • 需要在合并前检查或处理潜在的冲突。
